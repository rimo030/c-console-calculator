# Console-Calculator
C언어로 구현한 콘솔 계산기 입니다.


### 프로젝트 소개
C로 스택을 구현하고 이를 이용해 입력받은 수식의 결과를 출력합니다.


### 주요 기능
중위식을 입력받아 후위식으로 변환한 뒤 스택을 이용해 계산합니다.  

- 덧셈(+), 뺄셈(-), 곱셈(*), 나눗셈(/), 제곱(^)
- 대괄호([]), 중괄호({}), 소괄호(())
- 2자리 수 이상 계산 가능
- 괄호 오류 확인
- 사용가능한 문자 제한 확인

  
실행 결과는 [블로그](https://munak.tistory.com/117)에서 확인 가능합니다.
